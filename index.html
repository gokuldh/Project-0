IMPORTANT CONCEPTS

1. How The Web Works :

The web can be defined as communications between a bunch of computers based on the user inputs. When a user searches any domain name or IP address like www.udacity.com, their system shall send a HTTP Request to the respected Server. The server then locates the desired HTML page in it's directory and sends the data to the user's system where a Web Browser interprets the data and presents it on the user's system. Udacity has provided users with This Video to help the developers learn about these concepts.

2. HTML :

HTML stands for HYPER TEXT MARKUP LANGUAGE. The web consists of huge stacks of HTML documents. HTML documents contain Text Elements which determine "The Visual Content" as well as Markup Elements which refers to "How It Looks". HTML is THE Language of Web Development. Usually HTML documents are stored on servers, which provide the necessary documents to the users whenver a user acceses a webpage. The data is sent to the user's System which is furthur interpretted by the Web Browser in use, this data is then displayed to the user. Find the video provided by Udacity On HTML here.

3. Tags And Elements :

A HTML documents consists of various Elements. While creating a HTML document, the developer specifies the type of each elemts which is to be interpretted by the Browser using Tags. Some of the tags are <a> tag, <img> tag, <b> tag etc. This video explains the distinction well.

4. Why Computers are Stupid :

Computers are so called stupid systems because they interpret instructions literally. This makes them very unforgiving since a small mistake by a programmer can cause mny bugs in a program. they do whatever we make them to do. They just follows the instructions given to them.

5. Inline vs Block Elements :

HTML elements are either inline or block. Block elements form an "invisible box" around the consisting content. some of the inline elements are <b>, <em>, <img>, <span> etc. Example of block elements are <div>, <p>, <form> etc.
